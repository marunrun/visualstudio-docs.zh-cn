# <a name="idiaenumsourcefilesget_count"></a>IDiaEnumSourceFiles::get_Count
检索源文件的数目。

## <a name="syntax"></a>语法

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>参数
 pRetVal

弄返回源文件的数目。

## <a name="return-value"></a>返回值
 如果成功，将返回 `S_OK`;否则，将返回错误代码。
