# <a name="idiastackframeget_localsbase"></a>IDiaStackFrame::get_localsBase
检索框架的局部变量的基址。

## <a name="syntax"></a>语法

```C++
HRESULT get_localsBase ( 
   ULONGLONG* pRetVal
);
```

#### <a name="parameters"></a>参数
 `pRetVal`

弄返回局部变量的基址。

## <a name="return-value"></a>返回值
 如果成功，则返回 `S_OK`。 如果属性不受支持，则返回 `S_FALSE`。 否则，返回错误代码。
