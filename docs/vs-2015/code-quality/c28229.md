# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28229：批注不支持参数  
  
 此警告意味着某一批注使用了参数，但该批注未声明参数。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
