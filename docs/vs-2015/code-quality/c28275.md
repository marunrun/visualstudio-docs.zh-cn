# <a name="c28275"></a>C28275
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28275： Macro_value 的参数 \_ \_ 为 null  
  
 此警告意味着模型文件而不是所分析的代码中存在内部错误。 在没有参数的情况下调用 *macroValue* 函数。
