# <a name="c1254"></a>C1254
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1254：插件版本不匹配：版本与 PREfast 驱动程序的版本不匹配  
  
 当插件版本而不是所分析的代码存在内部错误时，代码分析工具将报告此警告。
