# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28211： SAL_context 需要静态上下文名称  
  
 此警告表明 `_Static_context_` 批注的操作数必须是工具定义的上下文的名称。 此警告在正常使用时不应发生。
