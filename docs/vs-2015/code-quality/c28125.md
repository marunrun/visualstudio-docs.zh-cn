# <a name="c28125"></a>C28125
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28125：必须从 try/except 块中调用函数  
  
 驱动程序正在调用一个函数，该函数必须从 try/except 块中调用，如[ProbeForRead](https://msdn.microsoft.com/library/windows/hardware/ff559876.aspx)、 [ProbeForWrite](https://msdn.microsoft.com/library/windows/hardware/ff559879.aspx)、 [MmProbeAndLockPages](https://msdn.microsoft.com/library/windows/hardware/ff554664.aspx)。  
  
## <a name="example"></a>示例  
 下面的代码示例将生成此警告：  
  
```  
ProbeForRead(addr, len, 4);  
```  
  
 下面的代码示例可避免此警告：  
  
```  
  
__try  
{  
    ProbeForRead(addr, len, 4);  
}  
__except(EXCEPTION_EXECUTE_HANDLER)  
{  
    Status = GetExceptionCode();  
    ... report error status  
}  
```
