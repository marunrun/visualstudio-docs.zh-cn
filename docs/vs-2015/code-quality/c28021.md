# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28021： \<param> 使用批注的参数 \<anno> 必须是指针  
  
 当批注对象不是指针类型时，将报告此警告。 此批注不能用于 `void` 或整型。
