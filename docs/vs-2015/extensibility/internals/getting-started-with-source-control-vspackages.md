# <a name="getting-started-with-source-control-vspackages"></a>源代码管理 VSPackage 入门
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

源代码管理 VSPackage 是源代码管理插件的更高级的替代方法。  (有关源代码管理插件的详细信息，请参阅 [创建源代码管理插件](../../extensibility/internals/creating-a-source-control-plug-in.md) 。 ) 源代码管理 VSPackage 允许完全控制源代码管理模型、功能和用户界面 (UI) ，并 [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 以 VSPackage 的形式集成到环境中。

 说明源代码管理解决方案的选择，并为选择适当的集成路径提供了广泛的准则。
