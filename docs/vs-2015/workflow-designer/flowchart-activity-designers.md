# <a name="flowchart-activity-designers"></a>流程图活动设计器
[!INCLUDE[wfd1](../includes/wfd1-md.md)] 包括多个系统提供的活动，这些活动在构造工作流时可以使用。 本节包含系统提供的活动中用于管理工作流中的流程图的活动。 以下主题介绍这些活动以及如何使用它们。

## <a name="in-this-section"></a>本节内容
 [流程图](../workflow-designer/flowchart-activity-designer.md) **用于创建定义和管理复杂流控制的工作流的流程图。**  [FlowDecision](../workflow-designer/flowdecision-activity-designer.md)根据是否满足指定的条件，为控制流提供两个替代项之一的分支。

 [FlowSwitch \<T >](../workflow-designer/flowswitch-t-activity-designer.md)当需要两个以上的备选分支时，根据匹配条件为控制流提供分支。
