# <a name="wpf-targets-files"></a>WPF .targets 文件

Windows Presentation Foundation (WPF) 通过添加一组特定于 WPF 的任务（任务被合并到一个特殊的 .targets  文件 Microsoft.WinFX.targets  ）来扩展 MSBuild。 此文件合并了在 WP 中生成 MSBuild 项目所需的一组 MSBuild 任务。
