# <a name="idebugportsupplierlocale2setlocale"></a>IDebugPortSupplierLocale2::SetLocale
设置端口供应商的区域设置。

## <a name="syntax"></a>语法

```cpp
HRESULT SetLocale(
   WORD wLangID
);
```

```csharp
int SetLocale(
   ushort wLangID
);
```

## <a name="parameters"></a>参数
`wLangID`\
要设置区域设置的标识符。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。
