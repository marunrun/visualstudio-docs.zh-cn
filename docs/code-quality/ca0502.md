# <a name="ca0502"></a>CA0502
不支持的语言

当用户尝试分析的 ASP.NET 项目包含一个或多个以非C#或 Visual Basic 语言编写的页面或类时，将发生此错误。 若要修复此错误，请从项目中删除该文件，或者禁用代码分析。
