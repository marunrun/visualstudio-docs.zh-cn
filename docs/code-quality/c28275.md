# <a name="c28275"></a>C28275
警告 C28275：@No__t-0Macro @ no__t-1value @ no__t-2 的参数为 null

 此警告意味着模型文件而不是所分析的代码中存在内部错误。 在没有参数的情况下调用*macroValue*函数。
