# <a name="c28022"></a>C28022
警告 C28022：此函数的函数类 \<classlist1 > 与用于定义该函数的 typedef 上的函数类 \<classlist2 > 不匹配。

 当批注中存在错误时将报告此警告。 Typedef 和函数本身都有 `_Function_class_` 批注，但它们不匹配。 如果两者都使用，则它们必须匹配。
