# <a name="javascript-object-expected"></a>缺少 JavaScript 对象
尝试将非 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 对象传递到需要对象的内置函数 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 。 各种内置函数需要中定义的对象 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] （而不是由宿主定义的对象或控件等外部组件）。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 确保作为参数传入的对象类型正确。
