# <a name="expected-var"></a>应有“var”
应输入 "var" （ERRnoVar 1034））  
  
 无法在需要时添加关键字 var。  
  
 有关 var 关键字的详细信息，请参阅[Var 语句](../../javascript/reference/var-statement-javascript.md)。
