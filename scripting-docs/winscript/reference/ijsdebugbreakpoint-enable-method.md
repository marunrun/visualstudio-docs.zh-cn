# <a name="ijsdebugbreakpointenable-method"></a>IJsDebugBreakPoint::Enable 方法
启用断点。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT Enable(void);  
```  
  
## <a name="return-value"></a>返回值  
  
## <a name="remarks"></a>备注  
 如果对已删除的断点调用，则返回 E_UNEXPECTED。  
  
## <a name="requirements"></a>要求  
 **标头：** jscript9diag
